## Описание проекта

**Weather App** — это  веб-приложение для отображения актуальной погоды в любой точке мира. Приложение предоставляет два способа поиска: по географическим координатам (широта и долгота) или по названию города/места. Пользователи могут создавать виджеты погоды в разных локациях, каждый из которых содержит интерактивную карту с точным местоположением. Проект выполнен без использования сторонних фреймворков и библиотек.

---

## Функциональные возможности

**Валидация данных:** Проверка корректности введенных пользователем координат (широта от -90 до 90, долгота от -180 до 180) и названий городов (минимум 2 символа).

**Асинхронные запросы:** Приложение использует `fetch` и `async/await` для получения данных о погоде через OpenWeatherMap API.

**Понятный код:**
  * Понятные названия переменных и функций (`validateLatitude`, `getWeatherByCoords`, `createWidget`).
  * Каждая функция или метод не превышает 25 строк.
  * Код структурирован по модулям: валидация, API запросы, UI, карты, координация.

---

## Дополнительные функции

 **Отображение дополнительной информации:** 
   *Температура (текущая и ощущаемая)
   *Влажность воздуха
   *Скорость ветра
   *Атмосферное давление
   *Название города и страны
   *Географические координаты

**Адаптивные иконки погоды:** Показываются разные иконки в зависимости от текущих погодных условий (солнце, облака, дождь, снег, туман, гроза и другие) через OpenWeatherMap API.

---

## Дополнительная функциональность

**Виджеты погоды:** Возможность добавлять неограниченное количество виджетов, отображающих погоду в разных местах. Каждый виджет независим и может быть удален кнопкой ×.

**Интеграция с картой:** Пользователи могут вводить широту и долготу или название города, чтобы получить погоду и просмотреть местоположение на интерактивной карте. Каждый виджет содержит собственную карту с маркером и popup с координатами. Поддерживается возможность добавления нескольких виджетов с картами.

**Два режима поиска:**
  * По координатам (широта и долгота)
  * По названию города (на русском или английском языке, с возможностью указания страны)

* **Кэширование данных:** Результаты запросов кэшируются на 10 минут для оптимизации использования API и ускорения повторных запросов.

* **Адаптивность:** Приложение адаптировано под все типы устройств (desktop, tablet, mobile).
